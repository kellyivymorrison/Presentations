# AI-generated scripts mentioned in the presentation

These are Python scripts generated from the prompts mentioned in the presentation.
They analyze various aspects of a code repository and generate Markdown files
with the results.

NOTE: these are quick and dirty programs. They may not be thorough: use them as
a basis for your own exploratory programs specific to your repo. Or, better yet,
generate your own using the prompts in the presentation as a starting point.

## analyze_config.py

Looks for configuration details: environment variables, command line options,
Docker configuration, etc.

## analyze_dependencies.py

Looks for dependencies: database calls, API calls, programs run in a shell, etc.

## analyze_security.py

Looks for security issues: unsecured endpoints, libraries with CVEs, etc.

## git_analyzer.py

Runs the git commands mentioned in the presentation.
